# Basic Chatbot

## 📌 Overview

A simple rule-based chatbot developed using Python for the CodeAlpha Python Programming Internship.

## ✨ Features

* Greets the user
* Answers predefined questions
* Responds to basic commands
* Exit using "bye"

## 🛠️ Technologies Used

* Python 3
* Functions
* If-Else Statements
* Loops

## ▶️ How to Run

```bash
python chatbot.py
```
